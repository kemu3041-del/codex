# 老客续费 Agent 分析报告工具

这是一个可直接预览的静态 HTML 工具 Demo，用于演示“输入客户名称和官网网址后，生成续费分析报告”的产品形态。

## 使用方式

直接用浏览器打开：

```text
customer-renewal-agent/index.html
```

核心流程：

1. 填写客户名称全称、官网网址、行业、续费窗口和已有服务范围。
2. 点击“生成续费分析报告”。
3. 查看 Agent 工作流状态、网站访问趋势、搜索收录、安全健康、营销动作和续费建议。
4. 点击“打印报告”可调用浏览器打印或另存为 PDF。

## 真实产品接入建议

- 网站使用情况：GA4、百度统计、SaaS 埋点、服务器日志。
- 用户访问情况：访问量、来源、页面停留、转化事件、移动端占比。
- 收录与 SEO：百度资源平台、Google Search Console、站点地图、关键词排名。
- 安全健康：HTTPS 证书、漏洞扫描、可用性监控、备份记录、性能检测。
- 全网营销：公众号、新闻稿、活动专题、招聘、ESG、品牌词搜索结果。
- 报告输出：HTML 报告、PDF、CRM 跟进记录、续费话术和服务包推荐。
