# Dental Clinic Homepage

基于 Behance 参考图复刻的牙科诊所官网静态页。

## 文件结构

```text
dental-clinic-homepage/
├── index.html
├── css/style.css
├── js/main.js
└── img/
```

## 实现说明

- `index.html`：页面主体内容，包含 Hero、数据数字、服务卡片、FAQ、医生案例、联系表单。
- `css/style.css`：冰蓝医疗视觉、牙种植体 CSS 图形、大字号排版、桌面与移动端响应式。
- `js/main.js`：滚动显现、菜单弹层、横向卡片切换、FAQ 单项展开、表单阻止默认提交。

## CMS/SaaS 字段建议

可替换字段包括：导航文案、菜单链接、Hero 标题、按钮链接、统计数字、服务卡片、FAQ 条目、医生案例、邮箱、地址、表单字段和页脚信息。

## 验证方式

直接打开 `index.html` 即可预览。建议检查桌面宽度 1440px 以上和移动端 375px，并测试 Menu、About 卡片箭头、FAQ 展开状态。
