# 04 阶段复盘

## 必须掌握

- 能正确使用 `useState`。
- 能解释 `useEffect` 适合处理什么。
- 能使用 `useRef` 保存 DOM 引用或非渲染数据。
- 能写一个简单自定义 Hook。
- 能清理定时器或事件监听。

## 自查问题

- 这个逻辑真的需要 `useEffect` 吗？
- 依赖数组是否完整？
- 是否有副作用没有清理？
- 重复逻辑是否可以封装为 Hook？

## 常见卡点

- 把派生数据也放进 state，导致状态不同步。
- 用 `useEffect` 修补本可以直接计算的逻辑。
- 忘记清理动画、监听或定时器。
