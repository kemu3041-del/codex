# 04 练习：Hooks 组织页面逻辑

## 练习 1：Tab 切换

任务：

- 使用 `useState` 实现 3 个 Tab。
- 点击 Tab 切换内容。

验收标准：

- 当前选中状态清晰。
- 不直接操作 DOM class。
- 内容由状态决定。

## 练习 2：模拟加载

任务：

- 使用 `useEffect` 模拟请求数据。
- 页面先显示 loading。
- 请求完成后显示列表。

验收标准：

- loading 状态正确。
- 组件卸载时清理计时器。
- 能处理空数据。

## 练习 3：封装 useToggle

任务：

- 封装 `useToggle`。
- 用它控制弹窗显示隐藏。

验收标准：

- Hook 文件独立。
- 返回当前状态和切换函数。
- 至少在两个地方复用。

## 常见错误

- `useEffect` 依赖数组漏写依赖。
- 在渲染期间直接触发 state 更新。
- 监听事件没有移除。

## 加餐挑战

封装 `useDebounce`，用于搜索输入防抖。
