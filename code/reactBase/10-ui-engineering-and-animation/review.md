# 10 阶段复盘

## 必须掌握

- 能在 React/Next.js 中组织样式。
- 能把静态动效拆成组件。
- 能正确初始化和清理 GSAP 动画。
- 能理解 Three.js 场景在组件中的生命周期。
- 能为移动端和性能做降级。

## 自查问题

- 动画是否依赖全局选择器？
- 组件卸载时是否清理动画？
- 移动端是否可读可操作？
- 动效是否影响页面主要信息？
- CMS 是否能替换主要文案和图片？

## 常见卡点

- 在 React 中直接复制传统全局动画脚本。
- 忘记清理 ScrollTrigger、事件监听或动画帧。
- 只在桌面尺寸验证视觉效果。
