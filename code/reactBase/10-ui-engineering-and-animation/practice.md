# 10 练习：React/Next.js 中的 UI 和动效

## 练习 1：响应式 Hero

任务：

- 创建产品 Hero 组件。
- 包含标题、说明、按钮和视觉媒体区域。
- 适配桌面和移动端。

验收标准：

- 文案不溢出。
- 移动端布局合理。
- 组件 props 可配置主要内容。

## 练习 2：GSAP 入场动画

任务：

- 给 Hero 标题、按钮和媒体区域增加入场动画。
- 动画在组件挂载后执行。
- 组件卸载时清理动画。

验收标准：

- 不直接污染全局选择器。
- 动画不会重复叠加。
- 低性能设备仍可正常阅读内容。

## 练习 3：滚动触发区块

任务：

- 创建 3 个内容区块。
- 滚动到对应区块时触发动画。

验收标准：

- 滚动动画节奏清晰。
- 移动端不出现遮挡。
- 动画失败时内容仍可见。

## 加餐挑战

把动画配置抽成 props，让同一个组件支持不同入场方向。
