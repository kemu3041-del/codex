# 03 阶段复盘

## 必须掌握

- 能区分 props 和 state。
- 能完成父传子和子传父。
- 能解释状态提升。
- 能写受控输入框。
- 能更新数组状态而不直接修改原数组。

## 自查问题

- 当前状态是否放在正确组件？
- 子组件是否做了不该做的数据修改？
- 表单值是否由 React 管理？
- 更新数组或对象时是否创建了新值？

## 常见卡点

- 忘记 React state 更新是异步批处理。
- 直接修改原数组导致页面不更新。
- 状态过早放到太高层，导致组件难维护。
