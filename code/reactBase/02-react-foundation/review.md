# 02 阶段复盘

## 必须掌握

- 能说明 `package.json` 里启动命令的作用。
- 能创建并导入组件。
- 能用 props 渲染不同内容。
- 能用数组渲染重复 UI。
- 能处理空状态。

## 自查问题

- 我是否还在复制多份重复 HTML？
- 我是否知道什么时候应该拆组件？
- 我是否能解释 `key` 为什么需要稳定？
- 我是否能看懂一个简单 React 项目的入口？

## 常见卡点

- import/export 语法不熟。
- JSX 中 `{}` 表达式和字符串写法混淆。
- 组件职责过大，所有内容堆在 `App`。
