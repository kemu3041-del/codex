# 02 练习：组件组合和列表渲染

## 练习 1：课程卡片组件

任务：

- 创建 `CourseCard` 组件。
- 接收课程标题、描述、难度、时长。
- 在页面中展示 3 个课程卡片。

验收标准：

- 数据从父组件传入。
- 子组件不写死课程内容。
- 组件文件命名清晰。

## 练习 2：数组渲染列表

任务：

- 准备一个课程数组。
- 使用 `.map()` 渲染课程列表。
- 每个列表项添加稳定 `key`。

验收标准：

- 不复制粘贴多份卡片 JSX。
- `key` 不使用随机数。
- 页面结构清晰。

## 练习 3：条件渲染

任务：

- 当课程数组为空时显示“暂无课程”。
- 当课程有内容时显示课程列表。

验收标准：

- 空状态和正常状态都能显示。
- 不通过手动操作 DOM 控制显示。

## 常见错误

- `.map()` 中忘记 `return`。
- `key` 使用数组索引后又进行删除或排序。
- 组件名小写导致 React 当成原生标签。
