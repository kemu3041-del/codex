# 09 练习：Route Handler 和表单提交

## 练习 1：GET 接口

任务：

- 创建 `/api/messages`。
- 返回留言列表。
- 前端页面请求并展示。

验收标准：

- 返回 JSON。
- 页面能展示数据。
- 空数据有提示。

## 练习 2：POST 表单

任务：

- 创建留言表单。
- 提交姓名和内容。
- 调用 POST 接口。

验收标准：

- 提交中按钮禁用。
- 成功后清空表单。
- 失败时显示错误。

## 练习 3：服务端校验

任务：

- 姓名不能为空。
- 内容不少于 5 个字。
- 校验失败返回错误信息。

验收标准：

- 前端和服务端都有基本校验。
- 服务端错误能显示到页面。
