# 09 阶段复盘

## 必须掌握

- 能创建 GET 和 POST Route Handler。
- 能读取请求体。
- 能返回 JSON 和错误状态。
- 能完成表单提交闭环。
- 能理解前端校验和服务端校验的区别。

## 自查问题

- 接口输入输出是否清楚？
- 表单提交是否有禁用和错误状态？
- 服务端是否做了必要校验？
- 客户端是否暴露了不该暴露的逻辑？

## 常见卡点

- 只做前端校验，忽略服务端校验。
- POST 接口没有处理非法输入。
- 表单提交成功后状态没有重置。
