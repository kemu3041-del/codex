# 01 阶段复盘

## 必须掌握

- 能说清楚 React 为什么不推荐直接操作 DOM。
- 能把简单 HTML 改成 JSX。
- 能用 `useState` 控制按钮、展开收起、选中状态。
- 能理解“状态变化驱动 UI 更新”。

## 自查问题

- 我是否还在想办法找 DOM 元素？
- 我是否知道某个 UI 文案由哪个状态决定？
- 我是否能把重复 HTML 拆成组件？
- 我是否能解释 JSX 和 HTML 的区别？

## 常见卡点

- JSX 语法看起来像 HTML，但它有 JavaScript 规则。
- React 事件处理函数传的是函数，不是字符串。
- 普通变量变化不会触发页面更新，state 才会。
