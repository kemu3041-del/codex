# 01 练习：从 DOM 操作改成状态驱动

## 练习 1：静态卡片改 JSX

任务：

- 准备一个普通 HTML 产品卡片。
- 改写成 React 组件 `ProductCard`。
- 保留标题、描述、价格、按钮和图片区域。

验收标准：

- JSX 中使用 `className`。
- 图片使用 `alt`。
- 组件名使用 PascalCase。

## 练习 2：收藏按钮

任务：

- 点击按钮后，在“收藏”和“已收藏”之间切换。
- 不允许使用 `querySelector`。
- 使用 `useState` 控制状态。

验收标准：

- 按钮文字由状态决定。
- 卡片样式可以根据状态变化。
- 多次点击状态不会错乱。

## 练习 3：显示隐藏详情

任务：

- 点击“查看详情”展开说明。
- 再次点击收起说明。

验收标准：

- 使用条件渲染。
- 不用手动设置 `display: none`。
- 按钮文案跟随状态变化。

## 常见错误

- 在 JSX 中写 `class` 而不是 `className`。
- 在点击事件里直接修改变量，不使用 state。
- 把 JSX 当 HTML 字符串拼接。

## 加餐挑战

把 3 个产品卡片渲染出来，并让每个卡片都有独立收藏状态。
