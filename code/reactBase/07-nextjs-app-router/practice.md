# 07 练习：App Router 页面结构

## 练习 1：课程中心

任务：

- 创建 `/courses` 页面。
- 展示课程列表。
- 点击进入 `/courses/[slug]`。

验收标准：

- 路由结构清晰。
- 详情页能读取动态参数。
- 找不到课程时显示 404。

## 练习 2：嵌套布局

任务：

- 给课程模块创建单独 layout。
- 在课程模块内显示二级导航。

验收标准：

- 课程模块布局不影响全站其他页面。
- 子页面内容正常插入。

## 练习 3：交互组件边界

任务：

- 在课程详情中添加收藏按钮。
- 收藏按钮作为 Client Component。
- 页面主体保持 Server Component。

验收标准：

- 只有需要交互的组件加 `"use client"`。
- 收藏按钮状态正常。

## 加餐挑战

为课程详情增加 `loading.js` 和 `error.js`。
