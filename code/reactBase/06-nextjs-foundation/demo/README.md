# Demo：Next.js 学习首页

本目录已经放置本章可运行 Next.js Demo。

已实现：

- 首页。
- 全局布局。
- 学习阶段卡片。
- 静态图片。

运行方式：

- 进入本目录：`cd 06-nextjs-foundation/demo`
- 安装依赖：`npm install`
- 启动开发服务：`npm run dev`
- 访问：`http://localhost:3000`

核心内容：

- `app/page.js` 首页。
- `app/layout.js` 全局布局。
- `app/globals.css` 全局样式。
- `components/LearningCard.jsx` 可复用组件。
