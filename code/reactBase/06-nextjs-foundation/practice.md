# 06 练习：第一个 Next.js 页面

## 练习 1：创建首页

任务：

- 创建 Next.js 首页。
- 页面包含标题、简介、学习阶段卡片。

验收标准：

- 首页路径为 `/`。
- 页面内容结构清晰。
- 样式不依赖外部平台。

## 练习 2：全局布局

任务：

- 创建全局 layout。
- 添加页头和页脚。
- 所有页面共享布局。

验收标准：

- 布局不会在每个页面重复写。
- 页面主体区域清晰。

## 练习 3：静态资源

任务：

- 在 `public/` 中放一张图片。
- 在页面中正确引用。

验收标准：

- 图片路径正确。
- 图片有 `alt`。
- 移动端不溢出。

## 加餐挑战

把学习阶段卡片拆成组件。
