# 08 练习：服务端和客户端数据请求

## 练习 1：服务端文章列表

任务：

- 在页面服务端请求文章列表。
- 渲染文章标题、摘要、发布时间。

验收标准：

- 首屏 HTML 包含文章内容。
- 没有把纯展示请求放到客户端副作用里。

## 练习 2：客户端搜索

任务：

- 添加搜索输入框。
- 用户输入后筛选或请求结果。
- 显示加载和空状态。

验收标准：

- 搜索组件是 Client Component。
- 页面主体仍可保持服务端渲染。
- 空状态文案清晰。

## 练习 3：环境变量

任务：

- 使用环境变量保存接口地址。
- 页面请求时读取配置。

验收标准：

- 不在代码里写死敏感配置。
- 能说明服务端变量和客户端变量的区别。
