# 08 阶段复盘

## 必须掌握

- 能区分服务端请求和客户端请求。
- 能根据页面场景选择渲染方式。
- 能处理加载、错误和空状态。
- 能使用环境变量。
- 能理解缓存和重新验证的基本作用。

## 自查问题

- 这个请求是否需要在浏览器发起？
- 首屏内容是否应该由服务端直接生成？
- 环境变量是否可能暴露给客户端？
- 缓存策略是否符合数据更新频率？

## 常见卡点

- 把所有数据请求都写进 `useEffect`。
- 在客户端暴露敏感接口配置。
- 忘记设计错误和空状态。
