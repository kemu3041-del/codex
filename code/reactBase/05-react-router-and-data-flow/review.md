# 05 阶段复盘

## 必须掌握

- 能理解客户端路由和普通页面跳转的区别。
- 能创建列表页和详情页。
- 能使用路由参数。
- 能用 Context 管理少量全局状态。
- 能完整处理异步请求状态。

## 自查问题

- 页面级组件和展示组件是否职责清楚？
- URL 是否能表达当前页面状态？
- loading、error、empty 是否都有 UI？
- Context 是否只用于真正跨层级共享的数据？

## 常见卡点

- 把所有数据都放进 Context。
- 详情页没有处理找不到数据。
- 请求失败时页面没有反馈。
