# 05 练习：路由和数据流

## 练习 1：列表页和详情页

任务：

- 创建文章列表页。
- 点击文章进入详情页。
- 详情页根据路由参数显示对应文章。

验收标准：

- URL 能表达当前文章。
- 刷新详情页后仍能显示正确内容。
- 找不到文章时显示错误或空状态。

## 练习 2：主题 Context

任务：

- 创建浅色和深色主题。
- 使用 Context 保存当前主题。
- 在导航和页面中读取主题。

验收标准：

- 不通过多层 props 传递主题。
- 切换主题后多个组件同步更新。

## 练习 3：异步状态

任务：

- 模拟文章接口请求。
- 展示 loading、error、empty 和 success。

验收标准：

- 每个状态都有明确 UI。
- 不出现页面空白。
- 错误状态可以重试。

## 加餐挑战

增加搜索参数，例如 `/articles?keyword=react`。
