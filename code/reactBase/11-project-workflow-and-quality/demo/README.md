# Demo：规范化项目模板

本目录已经放置本章可运行 Next.js Demo。

已实现：

- 清晰目录结构。
- 示例组件。
- 示例数据。
- 启动和构建说明。
- 组件说明文档。

运行方式：

- 进入本目录：`cd 11-project-workflow-and-quality/demo`
- 安装依赖：`npm install`
- 启动开发服务：`npm run dev`
- 构建验证：`npm run build`

目录重点：

- `app/`：页面和全局样式。
- `components/`：展示组件。
- `data/`：模拟数据。
- `docs/component-guide.md`：组件说明。
