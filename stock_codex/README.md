# stock_codex

股票分析工作区，用于按天沉淀资讯、市场局势分析、策略研究、数据看板和阶段性报告。

## 目录说明

- `daily/`：按日期保存每日资讯、盘面总结和观察清单。
- `market-research/`：宏观、行业、主题、公司研究。
- `strategies/`：交易规则、技术指标、回测和复盘。
- `data/`：行情、财报、资讯等数据文件及字段说明。
- `dashboards/`：分析看板和可视化 Demo。
- `reports/`：周报、月报、专题报告。
- `templates/`：常用分析模板。

## 每日文件命名

```text
stock_codex/daily/2026/06/2026-06-17.md
```

如果需要最新资讯，必须先核验来源和时间，再写入对应日期文件。
